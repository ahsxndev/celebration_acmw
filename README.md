## 🌟 The Importance of This Event

> **Empowering the Next Generation of Tech Leaders** > The **RCET Celebration of Women in Computing (RCETCWIC 2026)** is a flagship event dedicated to supporting, celebrating, and advocating for the full engagement of women in all aspects of the computing field. 

In an ever-evolving tech landscape, bridging the gender gap is critical. This event and its digital platform serve as a powerful catalyst for change by:
- 🛡️ **Dismantling Barriers:** Addressing biases and providing actionable solutions for women entering the tech workforce.
- 💡 **Fostering Innovation:** Hosting deep-dive workshops and keynotes on cutting-edge topics like Artificial Intelligence, Cybersecurity, and Smart Energy Systems.
- 🤝 **Building Community:** Creating a safe, empowering space for students and professionals to network, collaborate, and find mentorship.

---

## 👨‍💻 My Contribution

> **Lead Developer & UI/UX Engineer** > I took complete ownership of the digital presence for RCETCWIC 2026, aiming to create a modern, lightning-fast, and highly accessible web application that reflects the prestige of the ACM-W chapter.

**Key Technical Achievements:**
- 🏗️ **End-to-End Development:** Built the entire platform from scratch using **Flutter Web**, ensuring a unified codebase that scales flawlessly across Desktop, Tablet, and Mobile screens.
- 🎨 **UI/UX Engineering:** Implemented a custom theming engine utilizing the brand's vibrant purple and accent magenta, translating conceptual designs into a pixel-perfect reality.
- 📐 **Responsive Architecture:** Engineered adaptive grid systems (`Wrap`, `Flex`, and `LayoutBuilder`) so complex elements like Speaker Cards and the Event Timeline look perfect on any device.
- 🔀 **Advanced Routing:** Integrated `go_router` for seamless, deep-linkable, and SEO-friendly single-page application (SPA) navigation.

---

## ✨ Core Features

- 📱 **Fully Responsive:** Fluid layout that perfectly adapts to laptops, iPads, and smartphones.
- 📅 **Dynamic Schedule:** A beautifully structured timeline view of Day 1 and Day 2 events.
- 🎙️ **Speaker Directory:** Custom profile cards for Keynotes, Workshop Leads, and Panelists.
- 🖼️ **Interactive Media:** Pulsing animations for the poster competition and dynamic hover effects.
- 🔗 **Quick Registrations:** Direct integrations with Google Forms via `url_launcher` for seamless attendee onboarding.

## 🏗️ Quick Setup (Local Development)

1. **Clone the repository**
   ```bash
   git clone https://github.com/ahsxndev/celebration_acmw.git
   cd celebration_acmw

2.  **Install dependencies**

    ```bash
    flutter pub get
    ```

3.  **Run the app on Chrome**

    ```bash
    flutter run -d chrome
    ```
